# Myntra Functional Clone
## features:
1. User-Friendly UI for better Aethetics ,readability and understanding.
2. Add to bag button to add item to the the bag.
3. 'bag' Checkout page to view selected item.
4. "X" to remove item from the bag.
5. Dynamic page content that changes accordingly.
